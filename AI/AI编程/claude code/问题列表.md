## 登录无法访问问题修复
### 1.发现curl不通
curl -I https://api.anthropic.com 
### 2. 编辑~/.zshrc
```bash
1.~/.zshrc
2.//在最后加入自己的代理端口
export HTTP_PROXY="http://127.0.0.1:7897" 
export HTTPS_PROXY="http://127.0.0.1:7897" 
export ALL_PROXY="http://127.0.0.1:7897" 
export NO_PROXY="localhost,127.0.0.1"
3.source ~/.zshrc
```
